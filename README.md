# vuestore

> A Vue.js project   just for enjoying 

## Build Setup

``` bash
# install dependencies
bower install

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

# 项目介绍
  ### 便利店  convenience store
  ### 目的:了解以及熟悉VUE基本语法套路 
  ### 需求：
  * 路由功能
  * 产品列表 
    + 分类显示 最热  生活  家居 零食 电器...  
    + 顶部：便利店名称以及logo 
    + 底部:购物 购物车 订单 个人中心 
    + 内容区域:
         - 购物: tab栏目--  分类的类别
         - 购物车:我下单的产品列表
         - 订单：我的所有订单
         - 个人中心 :登陆 头像区域 收货地址 昵称 修改密码 意见联系我们
         - 产品详情页  购买数量  点击加入购物车
